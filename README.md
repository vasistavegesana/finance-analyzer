# Finance Analyzer

A Python project that analyzes personal expenses from a CSV file.

## Features
- Reads expenses from a CSV file
- Calculates total expenses
- Calculates remaining monthly income
- Groups expenses by category
- Demonstrates clean data handling

## Technologies Used
- Python
- CSV module
- Pandas
- Git & GitHub

## What I Learned
- Reading and cleaning CSV data
- Writing reusable functions
- Using dictionaries for aggregation
- Version control with Git
- Pandas operations
